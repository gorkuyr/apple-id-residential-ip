## 住宅IP和机房IP，差的不只是价格

很多人对"代理IP"的理解还停留在"换个国家的IP地址"这个层面，但对苹果、TikTok、Google这些平台来说，IP背后还藏着一层信息——这个IP是从哪个网络出来的。

简单分一下类：

1. **机房IP**：来自数据中心的服务器，价格便宜，量大，但平台一眼就能识别出"这是机房流量"
2. **动态代理IP**：每次连接换一个IP，看起来像普通用户，但同一个IP段可能被几千人共用过，黑历史一大堆
3. **住宅IP**：由真实的家庭宽带账号分配出来，ISP信息显示为本地运营商（比如AT&T），地理位置对应真实的居民区
